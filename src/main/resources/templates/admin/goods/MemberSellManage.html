위탁 판매 관리 시스템

UPCYCLING BUS : UPBUS CUSTOMERS CONSIGNMENT MANAGEMENT

1단계

신청 승인 단계
회원이 판매 신청한 승인 대기 상품 리스트입니다. 확인 후 승인 또는 거절 처리할 수 있습니다. 승인 처리 하는 경우 바로 승인 안내가 발송되며, 거절 처리 하는 경우 고객에게 거절 안내가 발송됩니다.
판매신청코드 ID 상품명 카테고리 판매신청가 판매수량 상품설명 신청일자 상품이미지 처리 단계를 한번 더 확인하세요

2단계

재고화 승인 단계
승인 처리된 상품 목록입니다. 배송 도착 상품의 검수를 먼저 진행하세요. 승인 처리 하는 경우 바로 재고로 등록되며, 거절 처리 하는 경우 고객에게 바로 거절 안내가 발송됩니다. 거절 처리한 경우 반송 프로세스를 진행하세요.
판매신청코드 ID 상품명 카테고리 판매신청가 판매수량 상품설명 신청일자 처리 단계를 한번 더 확인하세요

3단계

재고 등록 단계
검수 완료 후 재고를 등록합니다. 2단계에서 승인 버튼을 클릭하여 재고화할 상품의 신청 정보를 연동하세요.
판매신청코드 ID 상품명 카테고리 판매신청가 판매수량 상품설명 신청일자 3단계 처리를 완료하면 재고로 등록됩니다.
재고화 불가 상품 관리
재고화 불가 프로세스를 진행하세요. 상품을 재고화 할 수 없는 경우, 고객 정보를 확인하고 배송 업체와 연계하여 반송 처리를 진행하세요.
